TSP

    This program finds a solution for the traveling salesman problem using Prim's
    algorithm and Depth First Search.

Files

    randomCities -  Generates the solution for N randomly placed cities.  N can
                    be set by the first argument otherwise the default is 20.

    testPoints -    Simple test main class with fixed cities.

    GraphVertex -   Points used by other functions.

    MinSpanTree -   Generates a minimum spanning tree from an array list of verticies

    TSP -           Static function which generates a TSP route on an MST.

    Grapher -       Extends JPanel to display the generated items.
